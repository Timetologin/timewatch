# Stake Bros - אתר ניאון לחבר'ה 🎮

## איך מריצים?
1. מתקינים את כל התלויות:
```
cd client
npm install
cd ../server
npm install
```

2. מריצים את השרת:
```
node index.js
```

3. מריצים את ה-Client:
```
cd client
npm run dev
```

4. נכנסים לכתובת:
```
http://localhost:5173
```
